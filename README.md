# Rapid Crisis Response - AI Emergency Intelligence

This is my frontend project based on an AI-powered emergency response system for hotels and large properties. I created this project using HTML, CSS, and JavaScript to show how a smart crisis management dashboard can help during emergency situations.

The website includes a modern landing page, real-time crisis map UI, emergency SOS demo, AI assistant chat section, command center dashboard, predictive safety features, pricing plans, case studies, and contact section. I also added animations, live counters, canvas-based map visuals, and interactive buttons to make the project feel more realistic.

As a fresher developer, this project helped me practice responsive web design, UI styling, JavaScript DOM manipulation, canvas drawing, and creating a complete single-page website from scratch.

## Features

- Responsive single-page website
- Emergency response themed UI
- Animated crisis map using Canvas
- SOS demo overlay
- AI assistant chat interface
- Role-based dashboard section
- Modern animations and visual effects
- Built with HTML, CSS, and JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Canvas API
